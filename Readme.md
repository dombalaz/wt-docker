# Wt library in docker
This is a simple docker image for [Wt library](https://www.webtoolkit.eu/wt). Currently the image provides Wt version 4.3.1 on latest alpine. Wt is built with just mandatory requirements.
## Installing
```
docker pull mink0/wt:latest
```
